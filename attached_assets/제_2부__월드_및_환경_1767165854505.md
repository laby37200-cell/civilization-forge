# 4장. 월드 맵 시스템

## 4.1. 맵 구조: 육각형 타일 그리드 (Hex Grid)

### 4.1.1. 기본 구조

본 게임의 월드 맵은 문명(Civilization) 시리즈와 유사한 **육각형 타일 그리드(Hex Grid)** 시스템을 채택합니다. 육각형 타일은 사각형 타일에 비해 더 자연스러운 이동 경로와 균등한 인접 타일 수(6개)를 제공하여 전략적 깊이를 높입니다.

### 4.1.2. 맵 규모

| 항목 | 수치 |
| :--- | :---: |
| 총 타일 수 | 약 1,000개 이상 |
| 도시 수 | 100개 |
| 국가 수 | 20개 |
| 도시당 타일 수 | 7개 (중앙 + 주변 6개) |
| 자연 지형 타일 | 약 300개 |

### 4.1.3. 줌 레벨

플레이어는 맵을 확대/축소하여 볼 수 있습니다:

| 줌 레벨 | 배율 | 용도 |
| :--- | :---: | :--- |
| **축소** | 75% | 전체 맵 조망, 전략적 상황 파악 |
| **기본** | 100% | 일반적인 플레이 |
| **확대** | 150% | 세부 정보 확인, 전투 상황 파악 |

### 4.1.4. 스크롤

맵은 무제한 스크롤이 가능하며, 가장자리 래핑(지구본처럼 연결)은 적용되지 않습니다.

## 4.2. 세계지도 타일 표시 시스템

### 4.2.1. 실제 지리 기반 맵

본 게임의 월드 맵은 **실제 세계지도를 기반**으로 합니다. 문명(Civilization) 시리즈처럼 육각형 타일로 세계를 표현하며, 각 대륙과 국가의 위치는 실제 지리와 비슷하게 배치됩니다.

### 4.2.2. 타일 표시 정보

각 타일에는 다음 정보가 표시됩니다:

| 표시 요소 | 위치 | 설명 |
| :--- | :--- | :--- |
| **지형 아이콘** | 타일 중앙 배경 | 평야/산악/숲/사막 등 지형 표시 |
| **소유국 색상** | 타일 테두리 | 아군(파랑)/적군(빨강)/중립(회색)/동맹(초록) |
| **병력 아이콘** | 타일 좌상단 | 병력 존재 시 병과 아이콘 표시 |
| **병력 수** | 병력 아이콘 옆 | 숫자로 병력 수 표시 |
| **건물 아이콘** | 타일 우상단 | 건물 존재 시 건물 아이콘 표시 |
| **자원 아이콘** | 타일 하단 | 특산물 존재 시 아이콘 표시 |
| **전투 표시** | 타일 전체 | 전투 중인 타일은 붉은 점멸 효과 |

### 4.2.3. 타일 상세 정보 팝업

타일을 클릭하면 다음 상세 정보가 팝업으로 표시됩니다:

```

+------------------------------------------+
| 지형: 숲 (Forest)                        |
| 소유: 한국 - 서울 (플레이어A)           |
+------------------------------------------+
| 병력: 350명                              |
|   - 보병: 200명                           |
|   - 궁병: 150명                           |
+------------------------------------------+
| 건물: 성벽 (Lv.1)                        |
|   - 방어력 +25%                          |
+------------------------------------------+
| 특산물: 목재 (50단위)                   |
|   - 건설 기간 -1턴                       |
+------------------------------------------+
| [병력 이동] [건물 건설] [공격]         |
+------------------------------------------+
```

### 4.2.4. 도시 표시 정보

도시 타일(중앙 타일)에는 다음 정보가 추가로 표시됩니다:

| 표시 요소 | 위치 | 설명 |
| :--- | :--- | :--- |
| **도시명** | 타일 중앙 상단 | 도시 이름 표시 |
| **나라명** | 도시명 아래 | 소속 국가 이름 |
| **영주 프로필** | 타일 좌하단 | 영주 아바타 이미지 |
| **도시 등급** | 도시명 옆 | 수도(왕관)/주요(성)/일반(집)/마을(오두막) |
| **인구 수** | 타일 우하단 | 도시 인구 표시 |

## 4.3. 맵 종류

### 4.3.1. 지구 고정 맵 (기본)

기본 맵은 실제 지구의 지리를 기반으로 한 고정 맵입니다. 각 도시는 실제 위치에 배치되며, 지형도 실제 지리를 반영합니다.

### 4.3.2. 랜덤 모드 (선택)

랜덤 모드를 선택하면 도시명과 국가 배치가 무작위로 섞입니다. 지형은 고정되지만, 어떤 국가가 어디에 배치될지는 매 게임마다 달라집니다.

# 5장. 타일 및 지형 시스템

## 5.1. 도시 클러스터: 중앙 타일 + 주변 6타일 구조

### 5.1.1. 도시 타일 구조

하나의 도시는 **7개의 타일 묶음**으로 구성됩니다:

```
        [N]
    [NW]   [NE]
       [C]
    [SW]   [SE]
        [S]

C: 중앙 타일 (도시 본체, 궁전/시청 위치)
N: 북쪽 타일
NE: 북동쪽 타일
SE: 남동쪽 타일
S: 남쪽 타일
SW: 남서쪽 타일
NW: 북서쪽 타일
```

### 5.1.2. 중앙 타일 (Center)

중앙 타일은 도시의 핵심입니다:

| 특성 | 설명 |
| :--- | :--- |
| **건물** | 궁전/시청이 위치 (파괴 불가) |
| **점령 기준** | 이 타일을 점령해야 도시 소유권 이전 |
| **병력 상주** | 기본 500명 상주 (궁전 방어) |
| **건설 불가** | 추가 건물 건설 불가 |

### 5.1.3. 주변 6타일

주변 6개 타일은 건물 건설 및 병력 배치가 가능합니다:

| 특성 | 설명 |
| :--- | :--- |
| **건물 건설** | 도시 등급에 따라 타일당 1~3개 건물 건설 가능 |
| **병력 배치** | 타일당 최대 500명 배치 가능 |
| **지형 효과** | 각 타일마다 다른 지형이 적용됨 |
| **방어선 구축** | 적의 침입을 막는 방어선 역할 |

## 5.2. 지형 종류 (8종)

### 5.2.1. 지형 목록

본 게임에는 8가지 지형이 존재합니다:

| 지형 | 이동 비용 | 방어 보너스 | 경제 효과 | 특수 효과 |
| :--- | :---: | :---: | :--- | :--- |
| **평야** | 1.0 | 0% | 식량 생산 +10% | 건설 및 이동 표준 |
| **초원** | 0.7 | 0% | - | 기병 이동력 보너스 |
| **산악** | 2.0 | +20% | 병력 생산 -10% | 공성병기 진입 불가 |
| **언덕** | 1.5 | +10% | - | 궁병 사거리/공격 보너스 |
| **숲** | 1.2 | +5% | 정찰 성공률 -20% | 매복 전략 유리 (기습) |
| **산림** | 2.0 | +10% | 정찰 불가 (숨김) | 깊은 숲, 시야 차단 |
| **사막** | 1.3 | +15% | 식량 생산 -15% | 소모전 유도 |
| **해안/바다** | 1.0 | 0% | 돈 생산 +20% (항구 시) | 해군/수송선만 이동 가능 |

### 5.2.2. 지형별 상세 설명

**평야 (Plains)**

평야는 가장 기본적인 지형입니다. 이동과 건설에 페널티가 없으며, 식량 생산에 약간의 보너스가 있습니다. 대부분의 수도와 주요 도시는 평야에 위치합니다.

**초원 (Grassland)**

초원은 이동이 빠른 지형입니다. 특히 기병의 이동력에 보너스가 적용되어, 기병 중심의 전략에 유리합니다. 방어 보너스는 없습니다.

**산악 (Mountain)**

산악은 방어에 매우 유리한 지형입니다. 방어력 +20% 보너스가 적용되며, 공성병기는 산악 지형에 진입할 수 없습니다. 단, 병력 생산에 -10% 페널티가 있습니다.

**언덕 (Hill)**

언덕은 중간 정도의 방어 보너스를 제공합니다. 특히 궁병이 언덕에 배치되면 사거리와 공격력에 보너스가 적용됩니다.

**숲 (Forest)**

숲은 은폐에 유리한 지형입니다. 정찰 성공률이 -20% 감소하며, 매복 전략을 사용할 때 기습 보너스가 적용됩니다.

**산림 (Deep Forest)**

산림은 숲보다 더 깊은 숲입니다. 정찰이 완전히 불가능하며, 시야가 차단됩니다. 방어 보너스도 숲보다 높습니다.

**사막 (Desert)**

사막은 식량 생산에 불리하지만, 방어에는 유리한 지형입니다. 소모전을 유도하는 데 적합합니다.

**해안/바다 (Coast/Sea)**

해안과 바다는 해군만 이동할 수 있는 지형입니다. 항구가 있는 도시에서만 바다 타일로 진입할 수 있습니다. 항구가 있으면 돈 생산에 +20% 보너스가 적용됩니다.

## 6.2. 도시 등급 시스템

### 6.2.1. 도시 등급 분류

도시는 4가지 등급으로 분류됩니다:

| 등급 | 도시 수 | 타일당 건물 한도 | 초기 병력 | 초기 돈 | 초기 식량 | 점수 |
| :--- | :---: | :---: | :---: | :---: | :---: | :---: |
| **수도** | 20개 | 3개 | 2,000 | 8,000 | 5,000 | +100 |
| **주요 도시** | 25개 | 2개 | 1,500 | 5,000 | 3,000 | +50 |
| **일반 도시** | 47개 | 1개 | 1,000 | 3,000 | 2,000 | +10 |
| **작은 마을** | 8개 | 1개 | 600 | 1,500 | 1,000 | -10 |

### 6.2.2. 등급별 특징

**수도 (Capital)**

수도는 각 국가의 정치적, 경제적 중심지입니다. 가장 많은 초기 자원과 병력을 보유하며, 타일당 최대 3개의 건물을 건설할 수 있습니다. 수도에만 건설할 수 있는 특수 건물(전략 사령부, 첩보 본부)이 있습니다.

**주요 도시 (Major City)**

주요 도시는 경제적, 문화적으로 중요한 도시입니다. 수도 다음으로 많은 자원을 보유하며, 타일당 최대 2개의 건물을 건설할 수 있습니다. 주요 도시에만 건설할 수 있는 특수 건물(국제 무역소)이 있습니다.

**일반 도시 (Normal City)**

일반 도시는 표준적인 도시입니다. 적당한 자원을 보유하며, 타일당 1개의 건물만 건설할 수 있습니다.

**작은 마을 (Small Town)**

작은 마을은 소규모 정착지입니다. 자원이 적고, 점수에서 오히려 -10점의 페널티가 있습니다. 하지만 전략적 요충지에 위치한 경우가 많아, 확장의 발판이 될 수 있습니다.

## 6.3. 도시 동적 성장 시스템

### 6.3.1. City Score (도시 점수)

도시는 게임 진행 중 성장하여 등급이 상승할 수 있습니다. 도시 점수(City Score)는 다음과 같이 계산됩니다:

```
City Score = 인구 + (건물 수 × 10) + (턴당 총 생산량 / 10)
```

### 6.3.2. 등급 승격 조건

| 현재 등급 | 승격 조건 | 승격 후 등급 |
| :--- | :--- | :--- |
| 작은 마을 | City Score 500 이상 | 일반 도시 |
| 일반 도시 | City Score 1,000 이상 | 주요 도시 |
| 주요 도시 | City Score 2,000 이상 + 수도 함락 | 수도 |

### 6.3.3. 수도 이전

원래 수도가 함락되면, 해당 국가의 가장 높은 City Score를 가진 주요 도시가 새로운 수도가 됩니다. 수도가 이전되면 기존 수도 전용 건물은 파괴되고, 새 수도에 건설해야 합니다.

## 6.4. 내전 처리

### 6.4.1. 내전 정의

같은 국가 내의 플레이어들이 서로 공격하는 상황을 내전(Civil War)이라고 합니다.

### 6.4.2. 내전 처리 방식

내전이 발생하면 해당 플레이어는 **새로운 국가로 취급**됩니다:

| 항목 | 처리 방식 |
| :--- | :--- |
| **국가명** | 새로 설정 가능 (예: 남한, 북한) |
| **외교 관계** | 원래 국가와 별개로 처리 |
| **수도 버프** | 내전 선포 시 공유 중단 |
| **동맹** | 원래 국가의 동맹 관계 해제 |

### 6.4.3. 내전 선포

내전을 선포하려면 다음 조건을 충족해야 합니다:

| 조건 | 설명 |
| :--- | :--- |
| **도시 수** | 최소 2개 이상의 도시 보유 |
| **병력** | 최소 3,000명 이상의 병력 보유 |
| **비용** | 돈 5,000 소모 |
