# 17장. 외교 시스템

## 17.1. 외교 관계 개요

### 17.1.1. 외교 관계 유형

본 게임에는 5가지 외교 관계가 존재합니다:

| 관계 | 설명 | 효과 |
| :--- | :--- | :--- |
| **동맹** | 가장 긴밀한 관계 | 공동 방어, 자원 공유, 정보 공유 |
| **우호** | 긍정적인 관계 | 거래 보너스, 비침략 |
| **중립** | 기본 관계 | 특별한 효과 없음 |
| **적대** | 부정적인 관계 | 거래 불가, 공격 가능 |
| **전쟁** | 교전 상태 | 모든 적대 행위 가능 |

### 17.1.2. 우호도 시스템

외교 관계는 우호도(0~100)에 따라 결정됩니다:

| 우호도 | 관계 |
| :---: | :--- |
| 80~100 | 동맹 가능 |
| 60~79 | 우호 |
| 40~59 | 중립 |
| 20~39 | 적대 |
| 0~19 | 전쟁 가능 |

## 17.2. 외교 행동

### 17.2.1. 동맹 제안

동맹을 제안하려면 다음 조건을 충족해야 합니다:

| 조건 | 설명 |
| :--- | :--- |
| 우호도 | 80 이상 |
| 비용 | 돈 2,000 |
| 수락 시 | 양측 모두 동맹 관계로 변경 |

### 17.2.2. 선전포고

선전포고를 하면 즉시 전쟁 상태가 됩니다:

| 항목 | 설명 |
| :--- | :--- |
| 비용 | 없음 |
| 효과 | 양측 모두 전쟁 상태로 변경 |
| 페널티 | 다른 국가와의 우호도 -10 (침략자 이미지) |

### 17.2.3. 휴전 제안

전쟁 중인 상대에게 휴전을 제안할 수 있습니다:

| 항목 | 설명 |
| :--- | :--- |
| 비용 | 협상에 따라 다름 (돈, 자원, 영토 등) |
| 수락 시 | 양측 모두 중립 상태로 변경 |
| 휴전 기간 | 최소 10턴 동안 공격 불가 |

### 17.2.4. 항복

패배가 확실한 경우 항복을 선언할 수 있습니다:

| 항목 | 설명 |
| :--- | :--- |
| 효과 | 모든 도시와 자원을 상대에게 양도 |
| 약탈 감소 | 약탈량 -80% |
| 게임 오버 | 항복한 플레이어는 게임 오버 |

## 17.3. 동맹 시스템 상세

### 17.3.1. 동맹 효과

동맹을 맺으면 다음과 같은 효과가 적용됩니다:

| 효과 | 설명 |
| :--- | :--- |
| **공동 방어** | 동맹국이 공격받으면 알림 수신 |
| **자원 공유** | 동맹국 간 자원 이전 가능 |
| **정보 공유** | 동맹국의 시야 공유 |
| **거래 보너스** | 동맹국 간 거래 시 수수료 면제 |

### 17.3.2. 동맹 해제

동맹을 해제하려면 다음 절차를 거쳐야 합니다:

| 항목 | 설명 |
| :--- | :--- |
| 통보 | 상대에게 동맹 해제 통보 |
| 유예 기간 | 5턴 동안 동맹 유지 |
| 해제 후 | 중립 상태로 변경, 우호도 -30 |

### 17.3.3. 동맹 배신

유예 기간 없이 동맹국을 공격하면 배신으로 간주됩니다:

| 항목 | 설명 |
| :--- | :--- |
| 페널티 | 모든 국가와의 우호도 -30 |
| 낙인 | 배신자 낙인 (10턴 동안 외교 페널티) |

## 17.4. 외교 채팅

### 17.4.1. 채팅 유형

| 유형 | 참여자 | 용도 |
| :--- | :--- | :--- |
| **1:1 채팅** | 두 플레이어 | 비밀 협상, 거래 |
| **동맹 채팅** | 동맹국 전체 | 전략 논의, 정보 공유 |
| **전체 채팅** | 모든 플레이어 | 공개 선언, 위협 |

### 17.4.2. 채팅 기능

| 기능 | 설명 |
| :--- | :--- |
| 텍스트 전송 | 자유로운 텍스트 메시지 |
| 거래 제안 | 채팅 내에서 거래 제안 가능 |
| 좌표 공유 | 지도 좌표를 클릭하여 공유 |
| 번역 | 자동 번역 지원 (선택) |

# 18장. 거래 시스템

## 18.1. 거래 개요

### 18.1.1. 거래 시스템 설계 철학

본 게임의 거래 시스템은 **PVP 직거래 방식**을 채택합니다. 플레이어 간 직접 협상을 통해 거래가 이루어지며, 시장 가격이나 수수료 개념은 적용되지 않습니다.

### 18.1.2. 거래 가능 품목

| 품목 | 거래 가능 여부 | 조건 |
| :--- | :---: | :--- |
| 돈 | 가능 | 제한 없음 |
| 식량 | 가능 | 제한 없음 |
| 특산물 | 가능 | 제한 없음 |
| **병력** | **가능** | 제한 없음 |
| **도시** | **가능** | 2개 이상 보유 시 |
| 건물 | 불가 | - |
| **평화협정** | **가능** | 전쟁 중일 경우 |
| **시야 공유** | **가능** | 제한 없음 |
| **첩보 포로** | **가능** | 포로 보유 시 |

## 18.2. 확장된 거래 품목 상세

### 18.2.1. 병력 거래

병력을 다른 플레이어에게 양도할 수 있습니다.

| 항목 | 설명 |
| :--- | :--- |
| **거래 단위** | 100명 단위 |
| **병과 지정** | 보병/기병/궁병/공성병기/해군 선택 가능 |
| **이동 시간** | 거래 체결 후 즉시 이전 |
| **제한** | 도시에 최소 100명 잔류 필요 |

**병력 거래 예시**

```
거래 제안:
- 제공: 보병 500명
- 요청: 돈 5,000 + 특산물(말) 50단위
```

### 18.2.2. 도시 거래

도시를 다른 플레이어에게 양도할 수 있습니다. 단, **2개 이상의 도시를 보유한 경우**에만 가능합니다.

| 항목 | 설명 |
| :--- | :--- |
| **거래 단위** | 도시 1개 |
| **제한** | 보유 도시 2개 이상 시만 가능 |
| **수도 거래** | 불가 (수도는 거래 불가) |
| **이전 시기** | 거래 체결 후 다음 턴부터 적용 |
| **포함 요소** | 도시 소유권, 소속 타일, 건물, 자원 모두 포함 |

**도시 거래 예시**

```
거래 제안:
- 제공: 부산 도시 (병력 1,200명, 건물 3개 포함)
- 요청: 돈 50,000 + 동맹 체결
```

### 18.2.3. 평화협정 거래

전쟁 중인 상대와 평화협정을 체결할 수 있습니다.

| 항목 | 설명 |
| :--- | :--- |
| **조건** | 전쟁 상태일 경우에만 가능 |
| **효과** | 전쟁 상태 해제, 외교 관계 "중립"으로 변경 |
| **휴전 기간** | 협정 후 10턴간 재선전포고 불가 |
| **위반 시** | 위반 국가 우호도 -50, 전체 국가 신뢰도 하락 |

**평화협정 예시**

```
평화협정 제안:
- 조건: 돈 10,000 지불 + 특산물(향신료) 100단위
- 추가 조건: 도시 1개 양도 (대구)
- 휴전 기간: 10턴
```

### 18.2.4. 시야 공유 협정

다른 플레이어와 시야를 공유하는 협정을 체결할 수 있습니다.

| 항목 | 설명 |
| :--- | :--- |
| **효과** | 상대방의 시야 범위 내 타일 정보 공유 |
| **기간** | 협정 시 지정 (10턴/30턴/무기한) |
| **해제** | 양측 합의 또는 일방적 해제 가능 |
| **일방 해제 시** | 우호도 -10 |

**시야 공유 예시**

```
시야 공유 협정 제안:
- 기간: 30턴
- 대가: 돈 3,000/턴 지불
```

### 18.2.5. 첩보 포로 교환

적국의 스파이를 생포한 경우, 해당 포로를 거래할 수 있습니다.

| 항목 | 설명 |
| :--- | :--- |
| **포로 획득** | 적 스파이 발각 시 사망 대신 생포 선택 가능 (50% 확률) |
| **포로 유지비** | 30 돈/턴 |
| **교환 대상** | 포로의 원래 소속 국가 |
| **교환 효과** | 포로 반환, 대가 획득 |

**첩보 포로 교환 예시**

```
포로 교환 제안:
- 제공: 중국 스파이 포로 1명
- 요청: 돈 5,000 + 우호도 +20
```

**포로 처형**

포로를 교환하지 않고 처형할 수도 있습니다.

| 항목 | 설명 |
| :--- | :--- |
| **처형 효과** | 포로 사망, 원래 국가와 우호도 -30 |
| **정보 획득** | 처형 전 심문으로 적 정보 일부 획득 가능 |

## 18.3. 거래 프로세스

### 18.3.1. 거래 제안

1. 거래 상대 선택
2. 제공할 품목 및 수량 입력 (다중 선택 가능)
3. 요청할 품목 및 수량 입력 (다중 선택 가능)
4. 추가 조건 입력 (평화협정, 시야공유 등)
5. 거래 제안 전송

### 18.2.2. 거래 수락/거절

거래 제안을 받은 플레이어는 다음 중 하나를 선택합니다:

| 선택 | 효과 |
| :--- | :--- |
| **수락** | 거래 즉시 체결, 품목 교환 |
| **거절** | 거래 취소 |
| **역제안** | 수정된 조건으로 재제안 |

### 18.2.3. 거래 체결

거래가 체결되면 다음과 같이 처리됩니다:

| 항목 | 처리 |
| :--- | :--- |
| 품목 교환 | 즉시 양측 자원에 반영 |
| 거래 기록 | 양측 거래 내역에 기록 |
| 우호도 | 양측 우호도 +2 |
| 행복도 | 양측 행복도 +2% (일시적) |

## 18.3. 특산물 가치 시스템

### 18.3.1. 기본 가치

특산물은 종류에 따라 기본 가치가 다릅니다. 이 가치는 거래 시 참고용으로 제공됩니다.

| 특산물 | 기본 가치 |
| :--- | :---: |
| 쌀/밀 | 15 |
| 해산물 | 20 |
| 비단 | 45 |
| 도자기 | 40 |
| 향신료 | 55 |
| 철광석 | 35 |
| 목재 | 25 |
| 말 (군마) | 70 |
| 금/보석 | 120 |
| 약재 | 50 |
| 차 (Tea) | 30 |
| 모피 | 45 |
| 소금 | 25 |
| 무기/화약 | 90 |
| 종이/문서 | 35 |

### 18.3.2. 가치 변동 요인

특산물 가치는 다음 요인에 따라 변동할 수 있습니다:

| 요인 | 가치 변동 |
| :--- | :--- |
| 희귀도 | 희귀할수록 가치 상승 |
| 수요 | 많은 플레이어가 원할수록 가치 상승 |
| 공급 | 많이 생산될수록 가치 하락 |
| 전쟁 | 군사 특산물 가치 상승 |

## 18.4. 거래 전략

### 18.4.1. 특산물 독점

특정 특산물을 독점하면 거래에서 유리한 위치를 점할 수 있습니다. 예를 들어, 말(군마)을 독점하면 기병을 원하는 국가에게 높은 가격을 요구할 수 있습니다.

### 18.4.2. 자원 봉쇄

적국의 자원 공급을 차단하여 경제적 압박을 가할 수 있습니다. 동맹국과 협력하여 특정 국가와의 거래를 금지하는 방식입니다.

# 19장. 첩보 시스템

## 19.1. 첩보 시스템 개요

### 19.1.1. 설계 철학

본 게임의 첩보 시스템은 v8.0의 스파이 시스템을 기반으로 하며, 정보 수집과 공작 활동을 통해 전략적 우위를 점하는 것을 목표로 합니다.

### 19.1.2. 첩보력 (Intelligence)

첩보력은 첩보 활동의 성공률에 영향을 미치는 수치입니다.

| 항목 | 수치 |
| :--- | :--- |
| 초기값 | 0 |
| 최대값 | 100 |
| 성장 | 턴당 +1 (대학 건물 시 +10%) |
| 효과 | 첩보 성공률 증가, 적 첩보 발각률 증가 |

## 19.2. 스파이 시스템

### 19.2.1. 스파이 생성

스파이는 외교관저 건물이 있는 도시에서 생성할 수 있습니다.

| 항목 | 수치 |
| :--- | :--- |
| 생성 비용 | 500 돈 |
| 생성 시간 | 2턴 |
| 유지비 | 50 돈/턴 |
| 최대 수 | 외교관저당 +1 |

### 19.2.2. 스파이 파견

스파이를 적 도시에 파견하여 첩보 활동을 수행합니다.

| 항목 | 설명 |
| :--- | :--- |
| 파견 비용 | 100 돈 |
| 이동 시간 | 거리에 따라 1~3턴 |
| 잠복 | 도착 후 자동으로 잠복 시작 |

### 19.2.3. 스파이 발각

스파이는 적에게 발각될 수 있습니다.

| 항목 | 확률 |
| :--- | :--- |
| 기본 발각률 | 3%/턴 |
| 첩보 본부 효과 | +3%/턴 (적 스파이 발각) |
| 발각 시 | 스파이 즉시 사망 |

## 19.3. 첩보 활동 종류

### 19.3.1. 정찰 (Reconnaissance)

적 도시의 정보를 수집합니다.

| 항목 | 수치 |
| :--- | :--- |
| 비용 | 50 돈 |
| 소요 시간 | 1턴 |
| 성공률 | 70% + (첩보력 × 0.3%) |
| 정보 | 병력 수, 건물 목록, 자원량 |

### 19.3.2. 전략 정찰 (Strategic Reconnaissance)

적의 전투 전략을 미리 파악합니다.

| 항목 | 수치 |
| :--- | :--- |
| 비용 | 200 돈 |
| 소요 시간 | 2턴 |
| 성공률 | 50% + (첩보력 × 0.3%) |
| 정보 | 적의 태세(Stance) 및 전략 텍스트 일부 |

### 19.3.3. 사보타주 (Sabotage)

적 도시의 건물을 파괴하거나 자원을 탈취합니다.

| 항목 | 수치 |
| :--- | :--- |
| 비용 | 300 돈 |
| 소요 시간 | 3턴 |
| 성공률 | 40% + (첩보력 × 0.3%) |
| 효과 | 건물 1개 파괴 또는 자원 20% 탈취 |

### 19.3.4. 선동 (Incitement)

적 도시의 행복도를 감소시킵니다.

| 항목 | 수치 |
| :--- | :--- |
| 비용 | 150 돈 |

## 19.4. 첩보 방어

### 19.4.1. 방첩 활동

첩보 본부 건물이 있으면 자동으로 방첩 활동이 수행됩니다.

| 항목 | 효과 |
| :--- | :--- |
| 적 스파이 발각률 | +3%/턴 |
| 첩보 활동 방해 | 적 첩보 성공률 -10% |

### 19.4.2. 암호 해독국 효과

| 항목 | 효과 |
| :--- | :--- |
| 전 도시 첩보 방어율 | +6% |
| 첩보 활동 가능 수 | +2 |

## 19.5. 첩보력 성장 시스템

### 19.5.1. 첩보력 성장 공식

```
턴당 첩보력 성장 = 1 + (대학 보너스) + (종이/문서 보너스)
```

### 19.5.2. 첩보력 성장 보너스

| 요인 | 보너스 |
| :--- | :--- |
| 대학 건물 | +10% |
| 종이/문서 특산물 | +20% |
| 첩보 활동 성공 | +2 (일회성) |

### 19.5.3. 첩보력 레벨별 효과

| 첩보력 | 효과 |
| :---: | :--- |
| 0~19 | 기본 첩보 활동만 가능 |
| 20~39 | 전략 정찰 가능 |
| 40~59 | 사보타주 가능 |
| 60~79 | 선동, 암살 가능 |
| 80~100 | 모든 첩보 활동 가능, 성공률 최대 |
