# 15장. 건물 시스템

## 15.1. 건물 시스템 개요

### 15.1.1. 설계 철학

본 게임의 건물 시스템은 v8.0의 20종 건물을 채택하되, v4.0의 수리/파괴 메커니즘을 적용합니다. 건물은 도시의 경제, 군사, 정보 능력을 강화하는 핵심 요소입니다.

### 15.1.2. 건물 분류

건물은 4가지 카테고리로 분류됩니다:

| 카테고리 | 건물 수 | 주요 역할 |
| :--- | :---: | :--- |
| **군사 시설** | 5종 | 병력 생산, 공격력/방어력 증가 |
| **경제 시설** | 5종 | 자원 생산 증가 |
| **정보/외교 시설** | 5종 | 첩보력 증가, 첩보 활동 지원 |
| **국가/행복도 시설** | 5종 | 행복도 증가, 특수 효과 |

## 15.2. 건설 규칙

### 15.2.1. 건설 위치

건물은 도시 중앙 타일을 제외한 주변 6개 타일에 건설할 수 있습니다.

### 15.2.2. 타일당 건설 한도

도시 등급에 따라 타일당 건설 가능한 건물 수가 다릅니다:

| 도시 등급 | 타일당 최대 건물 수 |
| :--- | :---: |
| 수도 | 3개 |
| 주요 도시 | 2개 |
| 일반 도시 | 1개 |
| 작은 마을 | 1개 |

### 15.2.3. 건설 제한

일부 건물은 특정 조건에서만 건설할 수 있습니다:

| 건물 | 건설 제한 |
| :--- | :--- |
| 전략 사령부 | 수도에만 건설 가능, 국가당 1개 |
| 첩보 본부 | 수도에만 건설 가능 |
| 국제 무역소 | 주요 도시 이상에만 건설 가능 |
| 항구 | 해안 타일에만 건설 가능 |

### 15.2.4. 건설 프로세스

1. 건설할 건물과 타일 선택
2. 건설 비용(돈) 지불
3. 건설 기간(턴) 동안 대기
4. 건설 완료 후 효과 적용

## 15.3. 수리 시스템

### 15.3.1. 건물 파괴 조건

건물은 다음 상황에서 파괴될 수 있습니다:

| 상황 | 파괴 확률 |
| :--- | :---: |
| 전투 패배 | 20% |
| 타일 점령 | 20% |
| 도시 점령 | 30% |
| 공성병기 공격 | 50% |
| 반란 | 20% |

### 15.3.2. 수리 비용 및 시간

파괴된 건물은 수리할 수 있습니다:

| 항목 | 수리 비용 | 수리 시간 |
| :--- | :---: | :---: |
| 기본 | 건설 비용의 40% | 건설 시간의 2/3 |

### 15.3.3. 수리 예시

무기고(건설 비용 3,000, 건설 시간 4턴)가 파괴된 경우:

```
수리 비용: 3,000 × 0.4 = 1,200 돈
수리 시간: 4 × 2/3 = 2.67 ≈ 3턴
```
