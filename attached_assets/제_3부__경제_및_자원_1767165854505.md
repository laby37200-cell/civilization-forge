# 7장. 자원 시스템

## 7.1. 4대 핵심 자원

### 7.1.1. 자원 개요

본 게임에는 4가지 핵심 자원이 존재합니다:

| 자원 | 역할 | 획득 방법 | 소비 방법 |
| :--- | :--- | :--- | :--- |
| **병력 (Troops)** | 전투 및 점령의 기본 단위 | 도시 생산, 징집 | 전투 손실, 유지비 |
| **돈 (Gold)** | 모든 거래 및 건설의 화폐 | 도시 생산, 거래 | 건설, 거래, 정찰 |
| **식량 (Food)** | 병력 유지 및 인구 성장 | 도시 생산, 거래 | 병력 유지비, 인구 유지 |
| **특산물 (Specialty)** | 행복도 및 거래의 핵심 | 도시 생산, 거래 | 소비, 거래 |

### 7.1.2. 병력 (Troops)

병력은 전투와 점령의 기본 단위입니다.

| 항목 | 수치 |
| :--- | :--- |
| **초기값** | 600~2,000명 (도시 등급별) |
| **최대값** | 도시 인구의 100% |
| **안전 징집량** | 도시 인구의 10% |
| **생산량** | 도시 기본 5~20명/턴 |
| **유지비** | 100명당 식량 7/턴 |
| **타일당 최대 배치** | 500명 |

**과도한 징집의 페널티**

도시 인구의 10%를 초과하여 징집하면 행복도가 감소합니다:

| 징집 비율 | 행복도 감소 |
| :--- | :---: |
| 10~20% | -5% |
| 20~30% | -10% |
| 30~50% | -20% |
| 50% 이상 | -30% + 반란 위험 |

**반란 시 병력 처리**

도시가 반란하면 해당 도시의 모든 병력은 적이 됩니다. 병력이 0명인 상태에서 반란이 발생하면, 해당 도시 인구의 50%가 병력으로 전환되어 반란군이 됩니다. 반란군은 도시 타일의 지형에 따라 보병/기병/궁병/해군으로 분배됩니다.

### 7.1.3. 돈 (Gold)

돈은 모든 거래와 건설의 화폐입니다.

| 항목 | 수치 |
| :--- | :--- |
| **초기값** | 1,500~8,000 (도시 등급별) |
| **최대값** | 무제한 |
| **생산량** | 도시 기본 40~150돈/턴 |
| **주요 용도** | 건설, 거래, 정찰, 첩보 |

### 7.1.4. 식량 (Food)

식량은 병력 유지와 인구 성장에 필수적인 자원입니다.

| 항목 | 수치 |
| :--- | :--- |
| **초기값** | 1,000~5,000 (도시 등급별) |
| **최대값** | 기본 10,000 (창고로 증가 가능) |
| **생산량** | 도시 기본 30~120식량/턴 (조정됨) |
| **주요 용도** | 병력 유지비, 인구 유지 |

**식량 부족 페널티**

식량이 부족하면 다음과 같은 페널티가 적용됩니다:

| 상황 | 페널티 |
| :--- | :--- |
| 식량 0이 되는 턴 | 행복도 -10% |
| 3턴 연속 식량 부족 | 매 턴 병력 2% 탈영 |
| 5턴 연속 식량 부족 | 매 턴 병력 5% 탈영 + 반란 위험 |

### 7.1.5. 특산물 (Specialty)

특산물은 행복도 유지와 거래의 핵심 자원입니다.

| 항목 | 수치 |
| :--- | :--- |
| **종류** | 15종 |
| **초기값** | 200~500단위 (도시 등급별) |
| **생산량** | 도시 기본 4~15단위/턴 |
| **주요 용도** | 행복도 증가, 거래, 특수 효과 |

## 7.2. 자원별 상세 정보

### 7.2.1. 자원 저장 한계

| 자원 | 기본 저장 한계 | 증가 방법 |
| :--- | :---: | :--- |
| 병력 | 도시 인구 100% | 인구 증가 |
| 돈 | 무제한 | - |
| 식량 | 10,000 | 창고 건설 (+50%) |
| 특산물 | 5,000 | 특산물 공방 건설 (+30%) |

### 7.2.2. 자원 이전

같은 국가 내의 도시 간에는 자원을 이전할 수 있습니다:

| 자원 | 이전 가능 여부 | 비용 |
| :--- | :---: | :--- |
| 병력 | 가능 (같은 국가만) | 이동 시간 소요 |
| 돈 | 가능 | 무료 |
| 식량 | 가능 | 무료 |
| 특산물 | 가능 | 무료 |

## 7.3. 병력 유지비 상세

### 7.3.1. 기본 유지비

병력 100명당 식량 7/턴이 소모됩니다. (기존 10에서 7로 조정)

### 7.3.2. 병과별 유지비

| 병과 | 유지비 (100명당) |
| :--- | :---: |
| 보병 | 식량 5 |
| 궁병 | 식량 6 |
| 기병 | 식량 8 |
| 공성병기 | 식량 10 |
| 해군 | 식량 12 |
| 첩보 | 돈 50 |

### 7.3.3. 유지비 계산 예시

수도에 병력 2,000명(보병 1,000, 기병 500, 궁병 500)이 있는 경우:

```
보병 유지비: 1,000 / 100 × 5 = 50 식량
기병 유지비: 500 / 100 × 8 = 40 식량
궁병 유지비: 500 / 100 × 6 = 30 식량
총 유지비: 50 + 40 + 30 = 120 식량/턴
```

# 8장. 생산 시스템

## 8.1. 턴별 최종 생산량 공식

### 8.1.1. 기본 공식

모든 자원의 최종 생산량은 다음 공식으로 계산됩니다:

```
최종 생산량 = 기본값 × (1 + 건물 보너스) × (1 + 지형 보너스) × 행복도 배율
```

### 8.1.2. 건물 보너스 (Building Bonus)

해당 도시에 건설된 모든 생산 관련 건물의 효과를 합산합니다.

예시: 시장(+40%) + 조세청(+50%) = 돈 생산 +90%

### 8.1.3. 지형 보너스 (Terrain Bonus)

도시 중심 타일의 지형에 따라 기본 생산량에 가산됩니다.

| 지형 | 병력 | 돈 | 식량 |
| :--- | :---: | :---: | :---: |
| 평야 | +10% | +5% | +10% |
| 산악 | -10% | 0% | -5% |
| 해안 | 0% | +20% | 0% |
| 사막 | -15% | 0% | -15% |

### 8.1.4. 행복도 배율 (Happiness Multiplier)

국민 행복도(0~100%)는 모든 생산량에 직접적인 배율로 작용합니다.

| 행복도 (%) | 생산 배율 | 특이사항 |
| :--- | :---: | :--- |
| **90~100** | 1.15 | 생산량 +15% (황금기) |
| **70~89** | 1.00 | 표준 생산량 |
| **50~69** | 0.85 | 생산량 -15% |
| **30~49** | 0.60 | 생산량 -40% (파업) |
| **0~29** | 0.30 | 생산량 -70%, 반란 위험 |

## 8.2. 도시 등급별 기본 생산량 (밸런스 조정 완료)

### 8.2.1. 조정된 기본 생산량

| 도시 등급 | 병력 (명/턴) | 돈 (Gold/턴) | 식량 (Food/턴) | 특산물 (Unit/턴) |
| :--- | :---: | :---: | :---: | :---: |
| **수도** | 20 | 150 | **120** | 15 |
| **주요 도시** | 15 | 100 | **90** | 10 |
| **일반 도시** | 10 | 70 | **60** | 7 |
| **작은 마을** | 5 | 40 | **30** | 4 |

(식량 생산량은 기존 대비 50% 상향 조정됨)

### 8.2.2. 생산량 계산 예시

서울(수도)의 턴당 돈 생산량 계산:

```
기본값: 150
건물 보너스: 시장(+40%) + 조세청(+50%) = +90%
지형 보너스: 평야(+5%)
행복도: 80% → 배율 1.0

최종 생산량 = 150 × (1 + 0.9) × (1 + 0.05) × 1.0
            = 150 × 1.9 × 1.05 × 1.0
            = 299.25 ≈ 299 돈/턴
```

# 10장. 시민 행복도 시스템

## 10.1. 행복도 결정 공식

### 10.1.1. 기본 공식

행복도는 다음 공식으로 계산됩니다:

```
행복도 = 기본값(50) + 증가 요인 - 감소 요인
```

### 10.1.2. 행복도 범위

행복도는 0~100% 범위로 제한됩니다.

## 10.2. 행복도 증가 요인

### 10.2.1. 특산물 보유

| 특산물 | 행복도 증가 |
| :--- | :---: |
| 쌀/밀 | +5% |
| 해산물 | +7% |
| 차 | +6% |
| 과일 | +6% |
| 술 | +5% |
| 향신료 | +8% |
| 금/보석 | +15% |
| 비단 | +5% |

### 10.2.2. 문화 건물

| 건물 | 행복도 증가 |
| :--- | :---: |
| 공원/극장 | +10% |
| 병원 | +5% |

### 10.2.3. 기타 증가 요인

| 요인 | 행복도 증가 |
| :--- | :---: |
| 전투 승리 | +3% (일시적) |
| 거래 성공 | +2% (일시적) |
| 평화 상태 (5턴 이상) | +5% |
| 세금 감면 | +5~10% |

## 10.3. 행복도 감소 요인

### 10.3.1. 세율

| 세율 | 행복도 감소 |
| :--- | :---: |
| 저세율 (50%) | 0% |
| 표준 (100%) | -5% |
| 고세율 (150%) | -15% |
| 착취 (200%) | -30% |

### 10.3.2. 전쟁 피로

| 상황 | 행복도 감소 |
| :--- | :---: |
| 전투 패배 | -5% (일시적) |
| 도시 포위 중 | -10%/턴 |
| 건물 파괴 | -3%/건물 |
| 병력 대량 손실 | -5~15% |

### 10.3.3. 자원 부족

| 상황 | 행복도 감소 |
| :--- | :---: |
| 식량 부족 | -10%/턴 |
| 특산물 부족 | -5%/턴 |
| 과도한 징집 | -5~30% |

### 10.3.4. 조세청 건물

조세청 건물은 돈 생산을 +50% 증가시키지만, 행복도가 -5% 감소합니다.

## 10.4. 행복도 구간별 상태 및 효과

### 10.4.1. 행복도 구간

| 행복도 | 상태 | 생산 배율 | 특수 효과 |
| :--- | :--- | :---: | :--- |
| **90~100%** | 황금기 | 1.15 | 모든 생산 +15%, 반란 불가 |
| **70~89%** | 호황 | 1.00 | 표준 생산, 반란 불가 |
| **50~69%** | 평범 | 0.85 | 생산 -15% |
| **30~49%** | 파업 | 0.60 | 생산 -40%, 반란 위험 낮음 |
| **0~29%** | 폭동 | 0.30 | 생산 -70%, 반란 위험 높음 |

### 10.4.2. 반란 발생 조건

행복도가 30% 미만이면 매 턴 반란 확률이 발생합니다:

| 행복도 | 반란 확률 |
| :--- | :---: |
| 20~29% | 5%/턴 |
| 10~19% | 15%/턴 |
| 0~9% | 30%/턴 |

### 10.4.3. 반란 발생 시 처리

반란이 발생하면:

| 항목 | 처리 |
| :--- | :--- |
| **도시 소유권** | 반란군(NPC)으로 이전 |
| **병력** | 기존 병력이 반란군이 됨 (0명이면 인구 50% 징집) |
| **건물** | 20% 확률로 파괴 |
| **자원** | 50% 약탈 |
