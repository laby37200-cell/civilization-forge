# 4. LLM Agent 학습 지침

## 4.1. 1~3번 파일 학습 후 이해해야 할 핵심 개념

- 턴 시스템의 동작 방식 (동시 진행, 명령 일괄 처리)
- 도시 7타일 클러스터 구조 (중앙 + 주변 6타일)
- 4대 핵심 자원 (병력, 돈, 식량, 특산물)
- 도시 등급 4종 (수도, 주요 도시, 일반 도시, 작은 마을)
- 지형 8종과 각각의 효과

## 4.2. 4번 파일 학습 후 이해해야 할 핵심 개념 (가장 중요)

- **전투력 = 능력치(70%) + 전략 점수(30%)**
- 전략 텍스트 입력 방식 (200자 이내, 미입력 시 0점)
- LLM API 호출/응답 데이터 구조
- 6종 병과와 상성 관계
- 타일 점령 조건 (적 병력 0명 + 아군 100명 이상)
- 도시 점령 시 모든 소속 타일 이전

## 4.3. 5~6번 파일 학습 후 이해해야 할 핵심 개념

- 건물 건설 규칙 및 효과
- 외교 관계 5단계 (동맹/우호/중립/적대/전쟁)
- 거래 가능 품목 (병력, 도시, 평화협정, 시야 공유, 첩보 포로 포함)
- 첩보 시스템 및 스파이 활동

## 4.4. 7~8번 파일 활용 방법

- 구체적인 수치가 필요할 때 참조
- 국가/도시/건물/병과 데이터 조회
- 밸런스 조정 내역 확인
- 용어 사전 참조
